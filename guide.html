Settings For Great Audio

Sound enhancements:

Turn this setting off If you are using external audio device, ex. DAC.

Software control:

To make sure no background software Is running, uncheck this setting. Leave It on for spatial audio.

  1. Go to control panel
  2. Change to small icons
  3. Click on Sound
  4. Click on the speaker you are using
  5. You will find the setting in the advance menu as shown on image below
  6. After unchecked, click apply.

Spatial Audio:

Some users prefer having spatial audio on, can improve in some scenario such as gaming or movies.
List of software recommended:

  • Dolby Atmos
  • DTS Sound Unbound
